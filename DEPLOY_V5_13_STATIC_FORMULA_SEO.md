# VA HOME v5.13 — Static Formula SEO

## Що змінено

- Блок «Формула VA HOME» перенесено з JavaScript у статичний HTML усіх 18 сторінок ароматів.
- Кожна сторінка містить власний текст задуму композиції, факти про об’єм, рекомендовану кількість паличок і налаштування інтенсивності.
- JavaScript більше не створює SEO-контент через `innerHTML`; він відповідає лише за компонування, взаємодію та GA4-подію переходу до Discovery Set.
- Оптимізовано `images/atmosphere/wood-shadow.webp`: 447 818 → 189 646 байт.
- Оптимізовано `images/collections/signature.webp`: 405 236 → 137 446 байт.

## Результат

- Унікальний контент доступний у першій HTML-відповіді без виконання JavaScript.
- Пошукові роботи одразу бачать зміст блоку та внутрішнє посилання на Discovery Set.
- Два важкі зображення разом стали легшими на 525 962 байти без помітної зміни якості.

## Розгортання

Завантажте вміст архіву на GitHub замість поточної версії сайту. Нових SQL-міграцій або Edge Function для цього SEO-етапу немає; архів уже містить функціонал попереднього релізу v5.12.
