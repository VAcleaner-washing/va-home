# VA HOME v5.15 — встановлення

## 1. Сайт

Завантажте весь вміст архіву в корінь сайту, замінивши попередні файли. Не завантажуйте лише окремі CSS/JS: v5.15 є повним релізом.

## 2. Supabase SQL

У Supabase → SQL Editor один раз виконайте:

`SUPABASE_MIGRATION_V5_15_REVIEW_PHOTOS.sql`

Міграція додає необов'язкове фото до відгуку та приватне завантаження через Edge Function. Наявні відгуки не змінюються.

## 3. Edge Function

Повторно розгорніть функцію:

`supabase/functions/submit-review/index.ts`

Для `submit-review` залиште Verify JWT вимкненим: функція сама перевіряє користувача, origin, тип і розмір фото та rate limit.

## 4. Перевірка

1. Відкрийте головну у приватному вікні та перевірте hero, телефон і Quick Add.
2. На сторінці аромату надішліть тестовий відгук із фото до 2 МБ.
3. В адмінці перевірте фото та схваліть відгук.
4. Після схвалення відгук має з'явитися на товарі й у блоці головної. Якщо схвалених відгуків немає, блок на головній навмисно прихований.
5. Проведіть тестове замовлення обома способами оплати.

Автоматичний лист після статусу `completed` і позначка «Перевірена покупка» вже входять у реліз та збережені без змін.
