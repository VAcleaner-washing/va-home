# VA HOME v5.12 — структура товару та етап довіри

## Що змінено

- Доказ формули та практичні акордеони винесені з правої колонки товару в окремий повноширинний блок.
- Прибрано порожню чорну зону під головним фото без sticky-ефекту та штучного розтягування зображення.
- На сторінці «Про VA HOME» додано живу історію походження бренду.
- Лист для статусу `completed` містить персональні посилання на відгук для придбаних ароматів.
- Discovery Set формує посилання на аромати, які входили до замовлення.
- Повторне запрошення для того самого виконаного замовлення блокується серверною позначкою.

## Порядок оновлення

1. У Supabase SQL Editor виконайте `SUPABASE_MIGRATION_V5_12_REVIEW_INVITATION.sql`.
2. Оновіть Edge Function `send-status-email` кодом із `supabase/functions/send-status-email/index.ts` та натисніть Deploy.
3. Для `send-status-email` залиште перевірку JWT відповідно до поточної адмін-конфігурації: функцію викликає авторизована адмін-панель.
4. Завантажте файли сайту на GitHub/хостинг.
5. На тестовому замовленні змініть статус на `completed` і перевірте:
   - лист прийшов один раз;
   - кнопки ведуть на правильні сторінки товарів і секцію відгуків;
   - повторне збереження не створює другого запрошення.
