# VA HOME v5.11 — Product Substance

Реліз продовжує v5.10 та додає індивідуальну продуктову аргументацію без порівнянь із конкурентами й без виправдовування ціни.

## Зміни

- 18 окремих текстів «Задум композиції», створених за нотами та характером формул.
- На товарній сторінці збережено короткий доказ процесу: аудит, витримка, контроль дифузії та комфорту.
- Додано природний перехід у Discovery Set для покупця, який хоче спочатку протестувати аромат.
- У GA4 додано подію `select_promotion` для переходу з блоку формули в Discovery Set.
- Склад і точні відсотки формул залишаються конфіденційними.

Міграція Supabase не потрібна. Публікуйте всі файли поверх v5.10 або v5.9.
