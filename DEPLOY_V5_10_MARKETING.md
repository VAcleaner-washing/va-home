# VA HOME v5.10 — Formula Story

Цей реліз побудований поверх v5.9 і не відкочує checkout, Discovery Set, відгуки, кабінет, wishlist, SEO чи GA4.

## Що змінилося

- На всіх 18 товарних сторінках додано компактний блок «Формула VA HOME».
- Рекомендації щодо паличок узгоджено з базою формул: 2–3 для ягідних композицій, 3–4 для решти.
- На головній та сторінці «Про VA HOME» додано чотири етапи процесу: архітектура, аудит, витримка, контроль партії.
- Виправлено стрибок внутрішнього скролу Discovery Set при виборі ароматів унизу списку.
- Оновлено SEO-описи головної та сторінки бренду.

## Важливо

- Міграція Supabase для цього релізу не потрібна.
- Секретні відсотки та склад формул на сайт не виводяться.
- GA4 ID залишається чинним.

## Публікація

Завантажте всі файли релізу на GitHub із заміною старих. Після публікації очистьте кеш браузера та перевірте товарну сторінку, Discovery Set і оформлення замовлення.
